An elevated container. Nest the variants to build hierarchy — each level sits one step further from the page background than the one around it.

Both the fill **and the corner radius** follow the active theme: the variants resolve to themed surface tokens, and `rounded-3xl` resolves through the per-theme radius scale. A Surface is tight under Moon and soft under Grass without knowing anything about either.

## Installation

Surface ships with the library — no separate install.

```tsx
import { Surface, Text } from 'panelui-native';
```

## Usage

```tsx
<Surface>
  <Text weight="medium">Account</Text>

  <Surface variant="secondary" className="mt-3">
    <Text size="sm" muted>Signed in as khalid@example.com</Text>

    <Surface variant="tertiary" className="mt-3">
      <Text size="xs" muted>Session expires in 12 days</Text>
    </Surface>
  </Surface>
</Surface>
```

## Examples

### The variant ladder

Each step is one level further from the background — nest them to build depth without hardcoding colours.

```tsx
<Surface className="gap-3 p-4">
  <Text>Default</Text>
  <Surface variant="secondary" className="gap-3 p-4">
    <Text>Secondary</Text>
    <Surface variant="tertiary" className="p-4">
      <Text>Tertiary</Text>
    </Surface>
  </Surface>
</Surface>
```

### Transparent

Keeps the padding and radius but paints no background — for a slot that should inherit whatever is behind it.

```tsx
<Surface variant="transparent" className="p-4">
  <Text>No background of its own.</Text>
</Surface>
```

## Variants

### `variant`

- `default` *(default)*
- `secondary`
- `tertiary`
- `transparent`

```tsx
<Surface variant="default" className="p-4">Default</Surface>
<Surface variant="secondary" className="p-4">Secondary</Surface>
<Surface variant="tertiary" className="p-4">Tertiary</Surface>
<Surface variant="transparent" className="p-4">Transparent</Surface>
```

Every part also accepts the underlying React Native props (`ViewProps` or `TextProps`) and a `className` for Tailwind utilities.

## Notes

`borderCurve: 'continuous'` is applied on iOS, giving Apple's squircle corner rather than a circular arc — visibly smoother at this radius. Android ignores it.

Use `variant="transparent"` when you want the padding and radius without a fill.
