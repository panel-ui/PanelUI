An on/off toggle, animated on the UI thread.

## Installation

Switch ships with the library — no separate install.

```tsx
import { Switch } from 'panelui-native';
```

## Usage

```tsx
<Switch value={enabled} onValueChange={setEnabled} />
```

## Variants

### `size`

- `sm`
- `md` *(default)*
