An indeterminate loading indicator. `Button` renders one for you when `loading` is set.

## Installation

Spinner ships with the library — no separate install.

```tsx
import { Spinner, Button } from 'panelui-native';
```

Or copy the source into your project, to own and edit it:

```bash
npx panelui-cli@latest add spinner
```

## Usage

```tsx
<Spinner />
<Spinner size="lg" />
```

## Examples

### Inline

```tsx
<Spinner size="sm" />
<Spinner />
<Spinner size="lg" />
```

### Centred in a screen

```tsx
<View className="flex-1 items-center justify-center">
  <Spinner size="lg" />
</View>
```

### Inside a button

You rarely need this — `Button`’s own `loading` prop inserts a variant-matched spinner for you.

```tsx
<Button loading={saving}>Save</Button>
```

## Variants

### `size`

- `sm`
- `md` *(default)*
- `lg`

```tsx
<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
```
