A picker that opens its options in a bottom sheet ‚Äî the right choice when the list is long or the surrounding layout cannot grow.

## Installation

Select ships with the library ‚Äî no separate install.

```tsx
import { Select } from 'panelui-native';
```

## Usage

```tsx
<Select
  value={fruit}
  onValueChange={setFruit}
  placeholder="Select a fruit"
  title="Favorite fruit"
>
  <Select.Item value="apple" label="Apple" />
  <Select.Item value="banana" label="Banana" />
</Select>
```

## Composition

```tsx
<Select>
  <Select.Item value="‚Ä¶" label="‚Ä¶" />
</Select>
```

- **`Select.Item`** ‚Äî One option. `value` identifies it, `label` is what is shown.

Use `InlineSelect` instead when the option list is short enough to expand in place.
