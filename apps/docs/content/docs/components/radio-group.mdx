A single-select list of options.

## Installation

RadioGroup ships with the library — no separate install.

```tsx
import { RadioGroup, Text } from 'panelui-native';
```

Or copy the source into your project, to own and edit it:

```bash
npx panelui-cli@latest add radio-group
```

## Usage

```tsx
<RadioGroup value={plan} onValueChange={setPlan}>
  <RadioGroup.Item value="free" label="Free — $0/month" />
  <RadioGroup.Item value="pro" label="Pro — $12/month" />
  <RadioGroup.Item value="team" label="Team — $36/month" />
</RadioGroup>
```

## Composition

```tsx
<RadioGroup>
  <RadioGroup.Item value="…" label="…" />
</RadioGroup>
```

- **`RadioGroup.Item`** — One option. `value` identifies it, `label` is what is shown.

## Examples

### Basic

One value for the group, set by whichever item is pressed.

```tsx
const [plan, setPlan] = useState('pro');

<RadioGroup value={plan} onValueChange={setPlan}>
  <RadioGroup.Item value="free" label="Free" />
  <RadioGroup.Item value="pro" label="Pro" />
  <RadioGroup.Item value="team" label="Team" />
</RadioGroup>
```

### Custom item content

Give an item children instead of a `label` when the row needs more than one line.

```tsx
<RadioGroup value={plan} onValueChange={setPlan}>
  {plans.map((p) => (
    <RadioGroup.Item key={p.id} value={p.id}>
      <View className="flex-1 gap-0.5">
        <Text weight="medium">{p.name}</Text>
        <Text size="sm" muted>{p.price} · {p.summary}</Text>
      </View>
    </RadioGroup.Item>
  ))}
</RadioGroup>
```

### Disabling the group or one option

```tsx
{/* The whole group. */}
<RadioGroup value={plan} onValueChange={setPlan} disabled={!canChangePlan}>
  {/* …items… */}
</RadioGroup>

{/* One option. */}
<RadioGroup.Item value="enterprise" label="Enterprise" disabled />
```
